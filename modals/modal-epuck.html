EPuck project

Microcontroller Control Signal Processing Path Planning C Real Life

Overview

This project involved programming an EPuck robot to navigate a tunnel autonomously. The robot was equipped with various sensors, including infrared sensors for obstacle detection and a microphone for sound detection.

Key Features

  • Algorithm to deduct object distance from IR sensor readings
  • PID to maintain the robot in the middle of the tunnel
  • Frequency analysis algorithm using FFT to make the rpbot reactive to certain frequencies

Technologies Used

  • C: Core logic and algorithm design
  • STM32: As the controlling chip of the system
  • IR Sensors: For object detection

Outcome

The resulting robot is capable of navigating through a tunnel autonomously and can react to specific sound frequencies. The robot can maintain its position in the middle of the tunnel using a PID controller and can detect obstacles using infrared sensors.
