End-to-End Planning for Autonomous Driving

Deep Learning Computer Vision Python Simulation

Overview

This project involved developing an end-to-end planning system for autonomous driving using deep learning and computer vision techniques.

Key Features

  • Neural Network using CNNs to extract information from an image
  • NN trained on simulated data using auxiliary task training, then fine tuned on real data

Technologies Used

  • Python: Core logic and algorithm design
  • Pytorch: For Deep Neural Network

Outcome

The resulting model is capable of predicting the future trajectory of a vehicle based on its past state and a dashcam image with a high degree of accuracy.

Next Steps

Further improvements include optimizing the model for real-time performance and integrating it with a full autonomous driving system.
