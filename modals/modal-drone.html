Drone Project

Optimization Path Planning Control Computer Vision Python Simulation Real Life

Overview

This project involved developing a computer vision and path planning system for a drone. The goal was to detect the position of the gates and optimize the drone's path to navigate through them efficiently. The project utilized Python for the implementation and included real-life testing of the drone's navigation capabilities.

Key Features

  • Gate detection in 3D space using a single RGB camera
  • Minimum jerk trajectory path planning
  • Smooth control logic for stability and accuracy

Technologies Used

  • Python: Core logic and algorithm design
  • OpenCV: For identifying gates in a single 2D frame
  • Numpy: For fast linear algebra

Outcome

The resulting control allows the drone to navigate through a series of gates with high precision and stability. The system was tested in both simulated and real-life environments, demonstrating its effectiveness in path planning and control.

Next Steps

Future improvements include on-board processing and increased speed for drone racing applications.
