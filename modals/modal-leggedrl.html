Quadruped RL project

Reinforcement Learning Control Python Simulation

Overview

This project involved training a quadruped robot using reinforcement learning techniques to navigate different terrains. The robot was trained in a simulated environment using PyBullet, focusing on learning to walk and trot effectively.

Key Features

  • Auadruped robot trained to walk using reinforcement learning
  • Central Pattern Generator used to aid in the stability of the robot

Technologies Used

  • Python: Core logic and algorithm design
  • Stable Baselines 3: For training the policy
  • PyBullet: For rendering

Outcome

The resulting model is capable of navigating various terrains with a high degree of stability and efficiency. The robot can walk and trot, demonstrating the effectiveness of reinforcement learning in robotic control.
