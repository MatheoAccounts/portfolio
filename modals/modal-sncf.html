SNCF Project: Shortest Path Finder

API Optimization Graph Search Python Simulation

Overview

This project involved developing an intelligent algorithm to compute the shortest path between two railway stations on a map. The project was completed in a simulated environment using ROS and Python, with the aid of visual and spatial data.

Key Features

  • Parsed a graph of stations and routes from map data
  • Implemented Dijkstra’s and A* algorithms for path planning
  • Integrated ROS for real-time simulation of the navigation system
  • Used camera feeds for potential station detection (CV integration)

Technologies Used

  • Python: Core logic and algorithm design
  • ROS (Robot Operating System): Simulation environment and real-time communication
  • OpenCV: For future integration with real-world station detection

Outcome

The resulting system is capable of dynamically computing optimal paths through a network of stations, with potential applications in logistics, rail automation, and drone-based monitoring systems.

Next Steps

Future improvements include real-world data integration, enhanced obstacle avoidance, and deploying the model on an embedded system for field testing.
