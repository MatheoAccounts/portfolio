SNCF Project: Shortest Path Finder

API Optimization Graph Search Python Simulation

Overview

This project involved developing an intelligent algorithm to compute the shortest path between two railway stations on a map. The project was completed using SNCF Open Data and involved parsing station and route data to create a navigable graph. The algorithm was implemented in Python, utilizing Dijkstra’s algorithms with early termination for efficient pathfinding.

Key Features

  • Parsed a graph of stations and routes from map data
  • Implemented Dijkstra’s algorithms with early termination for path planning
  • Used an API and a frontend and made use of GET requests

Technologies Used

  • Python: Core logic and algorithm design
  • Steamlit: For frontend
  • FastAPI: For backend

Outcome

The resulting app is capable of dynamically computing optimal paths through a network of stations, with potential applications in ticket sales and railway traffic optimization.

Next Steps

Future improvements include dynamic ticketing data integration, even better optimization, and making the system publically available.
