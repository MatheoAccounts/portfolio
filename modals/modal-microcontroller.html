Microcontroller Project

Microcontroller Assembly Real Life

Overview

This project involved programming a microcontroller (ATMEGA128) to act as a central unit for an intelligent house. Using assembly language, the microcontroller was programmed to control various sensors and devices, including a distance sensor, an LCD screen, a temperature sensor, and an IR remote for communication.

Key Features

  • Microcontroller programmed in assembly language
  • Various sensors used to emulate the different aspects of an intelligent house

Technologies Used

  • Assembly: Core logic and algorithm design
  • ATMEGA128: For frontend
  • Distance sensor: For detecting objects
  • LCD screen: For information display
  • Temperature sensor: For measuring temperature
  • IR remote: For communicating with the microcontroller

Outcome

The resulting microcontroller system is capable of controlling various aspects of an intelligent house, including object detection, temperature monitoring, and user interaction through an IR remote.
