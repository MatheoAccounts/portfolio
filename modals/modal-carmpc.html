Car Model Predictive Control Project

Optimization Control Matlab Simulation

Overview

This project focused on developing a model predictive control (MPC) for a car to navigate a road with another vehicle. Various MPC techniques were explored, including linear MPC, nonlinear MPC, robust tube MPC, and LQR.

Key Features

  • MPC to control a car
  • Auto speed adjusting, car passing, and car following
  • MPC theory such as terminal sets and robust invariant sets used to have robust and stable control

Technologies Used

  • Maltab: Core logic and algorithm design

Outcome

The resulting MPC is capable of controlling the car so that it can either follow, avoid, or pass the car in front of it.
