DEA Research Project

System Identification Reinforcement Learning Control Python Simulation Real Life

Overview

This project involved the characterization and control of a new type of DEA (Dielectric Elastomer Actuator). characterization was done using system identification techniques, and control was implemented using both reinforcement learning and PID.

Key Features

  • DEA characterization using a custom-made test bench, and then modeled the system using Supervised Learning on Recurrent Neural Networks (RNN)
  • Different control using self-sensing were investigated such as PID and Reinforcement Learning

Technologies Used

  • Python: Core logic and algorithm design
  • Oscilloscope and DAQ: For testing and system identification
  • Pytorch: For Reinforcement Learning and RNN

Outcome

The project resulted in showing that the capacitive sensing of the DEA changed too much with the voltage applied to the DEA, making it complex to use for control. However, the system identification and control techniques developed can be applied to other types of actuators.
