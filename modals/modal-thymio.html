Thymio Project

Path Planning Computer Vision Control Python Real Life

Overview

This project involved programming a Thymio robot to navigate an environment autonomously. The robot was equipped with various sensors, including infrared sensors for obstacle detection and a camera for visual input. The project focused on implementing path planning algorithms and integrating computer vision techniques to enhance the robot's navigation capabilities.

Key Features

  • Obstacle detection using Computer Vision
  • Dijkstra’s algorithms for path planning
  • Local obstacle avoidance
  • Pose estimation using Kalman Filter

Technologies Used

  • Python: Core logic and algorithm design
  • OpenCV: For obstacle detection
  • IR sensors: For local obstacle avoidance

Outcome

The resulting robot is capable of navigating through an environment autonomously, avoiding obstacles, and planning paths using Dijkstra’s algorithm.
