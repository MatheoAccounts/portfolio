Anthill Simulation Project

Simulation Object Oriented Programming C++ Simulation

Overview

This project involved developing a simulation of an anthill using object-oriented programming principles in C++. The simulation models the behavior of ants within an anthill, including their interactions and environmental factors.

Key Features

  • Object-oriented design to model ants and their environment
  • Simulation of ant behavior and interactions
  • Graphical representation of the anthill and ant movements using gtkmm3
